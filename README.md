# Hardware calc macro written in verilog

These are 16-bit arithmetic circuits in various formats.

## Arithmetic type  

- Add
- Sub
- Mul
- Div
- Rem(Only int)

## Implement Format  

- int16
- fix16
- bf16 (Not yet verified)
